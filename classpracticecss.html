Introduction to CSS

Today has been an introduction to CSS. We have learnt different ways to style our HTML elements. We have learnt how to use the style attribute to add CSS properties directly to HTML elements. We have also learnt how to use the style tag to add CSS rules that apply to multiple elements on the page. We have also learnt about different CSS properties such as color, font-size, background-color, and border. We have also learnt about CSS selectors and how they can be used to target specific HTML elements for styling.
